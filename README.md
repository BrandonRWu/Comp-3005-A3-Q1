Execution - 
1. You will need to have Maven downloaded to run this app. Maven can be downloaded from 'https://maven.apache.org/download.cgi' and ensure that you download maven and not maven daemon.
2. Open command prompt and move to the directory of the unzipped folder
3. To create the table run 'psql -U postgres -f create_students_table.sql' in the command prompt. This will create the database and table the program will interact with
4. To run the program, run 'mvn exec:java -Dexec.mainClass=org.example.Main'
